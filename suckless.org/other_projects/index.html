Other projects

There are several other projects which are inspired by the spirit of suckless.

  • alt - abstract language tree
  • autonet - automatic wifi network chooser for OpenBSD
  • bgs - background setter
  • bug - commandline todo-tracking system
  • bus - daemonless message broadcasting IPC
  • cross-chroot - cross platform qemu-powered chroot utilities (based on the ideas of the maemo sdk)
  • dietline - minimalist implementation of a readline-like library
  • dmc - dynamic mail client
  • dvtm - screen-like dwm (curses)
  • dzen - display status/notification/menu windows
  • gendoc - a suckless documentation generation alternative to Shpinx and ReadtheDocs
  • honden - flashcard system built on top of oboeta
  • ired - minimalistic hexadecimal editor inspired in radare
  • kelp - source code annotation framework
  • librarian - less sucky alternative to pkg-config
  • maan - various projects which adhere to the suckless philosophy
  • mdslides - pure HTML+CSS slide presentations in Markdown
  • micy - minimalistic mouse handler (depends on driver)
  • mosaic - UNIX style music tools
  • nt - 'at' wrapper for setting one time notification alarms.
  • oboeta - minimalistic plain-text flashcard system
  • pcw - popup chat windows (ii front end)
  • ptar - plain-text archives: like tar, but better
  • sat - simple incompatible alternative to at
  • sdhcp - tiny dhcp client
  • sj - simple jabber: modular FIFO-based XMPP client
  • slpm - suckless package manager
  • sltar - suckless tar
  • smatrix - cmatrix clone with improved efficiency and compatibility
  • smu - simple markup (see also this fork with improved markdown support)
  • spp - simple pre processor
  • spt - simple pomodoro timer
  • srw - simple read wrapper
  • ssfn - single header bitmap, pixmap, vector font renderer
  • ssg - slide generation system with markdown-like syntax
  • ssg - Static site generator 180 LoC
  • sup - minimalistic sudo replacement
  • sw - minimalistic web framework
  • sxmo - minimalistic phone environment which uses suckless tools
  • tiv - terminal image viewer
  • wjt - slider widget for X
  • xbmouse - simply bind a mouse button to a command
  • xkev - simply simulate KeyPress/KeyRelease
