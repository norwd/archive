KD7DVD's FT-221R

This is a radio purchased by my friend, Chris, KD7DVD recently, so that we could exchange 2m CW in order to get our code speed up for the General 13 WPM code test. Naturally, being an older radio, there were no pictures of it anywhere on the web, that we could find. So, we decided to remedy that situation.

The images of the inside top are impressive mostly because Yaesu did such a nice job of labeling and sorting out all the adjustments you'd ever want to make while performing alignment. I was very impressed the first time I saw it, as was Chris.
