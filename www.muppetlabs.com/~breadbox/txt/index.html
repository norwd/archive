Random Bits of Text

The links on this page lead to texts, otherwise unrelated, which may or may not have the potential to be engaging.

  • The Teensy Files
    A series of essays about why Linux ELF executables are as big as they are, and how to make them smaller.
  • My Own Private Binary: An Idiosyncratic Introduction to Linux Kernel Modules
    An examination of loadable kernel modules on Linux, and how they can be used to create a custom binary executable format.
  • Bootstrapping Understanding: An Introduction to Reverse Engineering
    This essay provides an in-depth walk-through of the process of reverse engineering a set of data files.
  • Cryptic Crossword: Amateur Crypto and Reverse Engineering
    An essay that presents the story of reverse engineering an unpublished scrambling algorithm.
  • Prime Number Hide-and-Seek: How the RSA Cipher Works
    This page describes in detail the mathematics that make the RSA cipher tick.
  • A Monovocalic Sonnet on Dante's "Inferno"
    This is an English "translation" of one of Giuseppe's Varaldo's monovocalic sonnets.
  • Short Words to Explain Relativity
    An explanation of Albert Einstein's Theory of Relativity entirely using words no more than four letters long.
  • Explaining the Monty Hall Problem
    A hand-curated collection of various explanations for a well-known, counterintuitive puzzle.
  • Measuring The Space Needle
    An essay on measuring heights using no special tools, just special triangles.
  • Making a Unique Keyboard
    A photo-essay describing the process of creating a keyboard with a personalized set of symbols.
