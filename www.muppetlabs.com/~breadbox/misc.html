Miscellaneous Non-textual Stuff

Most of my activity finds expression in textual media, but there are occasionally exceptions. I've collected some of the exceptions here.

  • My Cryptic Sketchbook
    An opaque little codex that seems to be about … something.
  • The Cynical-Literalist's Tarot
    A hand-drawn Major Arcana (of sorts).
  • Fish Fillets Solution Archive
    Fish Fillets is a difficult puzzle game, originally written by Altar Interactive and now available under the GPL.
  • The Songfight Cover Bot
    An automatic process that generates cover images for songfight.org.
  • My Linux console font
    Some of the uglier characters in the default console font prompted me to create my own. In addition to the usual support for the Latin-1 character set and the VT100 special characters, this font also covers the full set of Latin-5, Latin-9, Windows-1252, and for good measure a few of the more frequently used characters that only appear in IBM codepage 437.
