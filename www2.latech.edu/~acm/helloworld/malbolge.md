Malbolge
========

------------------------------------------------------------------------

The author of \'Malbolge\' (don\'t know why the missing \'e\', maybe to
make an 8-char file name) has made it harder by having the spec and the
source of the interpreter different \-- the actions of the input and
output instructions are swapped. I guess that\'s right in keeping with
the stated object of being hard to use. So there are two versions, the
first complies with the spec, the second with the source code.
