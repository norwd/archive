Simplyread

Description

A piece of javascript to remove distractions from webpages, similar (but much more simple) to readable. Download the source from the webpage, then install by running this:

cat keybind.js simplyread.js >> $HOME/.surf/script.js

The default keybinding is ctrl-alt-r.
