R9P

Author:  Kris Maglione
License: MIT
Version: 0.4

r9p is a simple 9P client implementation for Ruby. It currently supports basic filesystem operations, including reading, writing, and listing files. It also supports the operations of the IO class by providing a true file descriptor through a proxy thread.
