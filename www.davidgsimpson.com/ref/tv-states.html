TV SHOWS SET IN EACH STATE

Shown here are examples of TV shows (mainly prime-time comedies and dramas) that are regularly set in various states.

Not included here are shows that are not set in any state (e.g. Gilligan's Island or Star Trek), shows that are set in a state whose name is never disclosed (e.g. Green Acres or Benson), shows that are set in a different location each episode (e.g. Route 66 or The Incredible Hulk), or any “reality” shows or animated shows. Shows set in a fictional town of a real state (e.g. Mayberry, North Carolina) are acceptable.
