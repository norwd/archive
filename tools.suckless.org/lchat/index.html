lchat

lchat (line chat) is a line oriented front end for ii-like chat programs. It handles the input from keyboard and output file in parallel. Thus, you are able to type messages while new chat lines are arriving. Its main focus is on usability and simplicity.

You can use lchat as frontend for the following programs and protocols:

  • ii for IRC
  • ratox for Tox
  • sj and jj for XMPP
  • icb for ICB

Requirements

  • libgrapheme
  • tail(1)
  • grep(1)

Download

  • lchat 1.0 (2022-12-28)

Development

You can browse its source code repository or get a copy using the following command:

git clone https://git.suckless.org/lchat
