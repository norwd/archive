dyn-battery

Description

This patch implements a function that displays the status and capacity of all present batteries (only supported for Linux).

The order of the format identifiers is hard-coded: battery number (%u), state (%s), capacity (%s). Example:
"[BAT%u: %s%s]" -> [BAT0: +58]
